DailyQuo is a cross-platform mobile application that delivers curated daily quotes with search, favorites, and author exploration features.

The project focuses on mobile application architecture, user authentication, and cloud-based data management using Firebase.

---

## Tech Stack
- Mobile: React Native, Expo
- Backend Services: Firebase Authentication, Firestore
- Language: JavaScript

---

## Key Features
- Daily quote delivery
- Search quotes by author or content
- Favorite and save quotes
- Author discovery and profile pages
- User authentication and profile management

---

## Architecture & Design

### Application Architecture
- Component-based React Native structure
- Centralized state and navigation handling
- Separation of UI components and data logic

### Backend Services
- Firebase Authentication for user management
- Firestore as a cloud-hosted NoSQL database
- Secure read/write rules for user-specific data
